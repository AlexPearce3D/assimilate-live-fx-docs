# Import Media and Maintain Folder Structure

There are many ways to import media files and you may want to maintain your folder structure. Here is a video tutorial that might be helpful:

> Video/resource: [https://youtu.be/87fHxVKVr0I](https://youtu.be/87fHxVKVr0I)

!!! warning

```
If you are using[ h.264 media, make sure to set up Assimilate to decode it properly!!](playback-h.264-files.md)
```

## Import all media to one timeline

1. Click on Import Clips
2. Select "Select Folder" and "Create New Timeline" and select the depth (10 will look 10 folders deep for all media).<br>
3. Under the Clip Import Options, for the Timeline use either "New From Foldername" to create a new timeline with the selected folder as the timeline name.\
   Or select Timeline:Current to import the clips into the current timeline.

All the files are now imported to one long timeline.

## Import each subfolder as it's own Timeline

1. Click on Import Clips
2. Select "Select Folder" and "Create New Timeline" and select the depth (10 will look 10 folders deep for all media).<br>
3.  This time, in the Clip Import Options, select Timeline: New Per Subfolder<br>

    This will create a new timeline for each subfolder, for example, there is a timeline for the folder 001\_Fire, a timeline for the folder 002\_Water, etc.<br>

## Import each subfolder as it's own shot with media as Versions.

1. Add a New Timeline
2. Click Import Clips
3.  This time, make sure "Select Folder" is enabled, but NOT "Create New Timeline". Set the depth to 1 to only go 1 folder deep.<br>

    This creates a slot for each folder, with the files inside as "Versions" that stack above.
