# How to Sample Multiple Video Sources through Videowall

Sometimes it is helpful to have multiple sources to sample lighting from, for example, if you want to sample from a fire video and a separate water video.

There are many different ways to achieve this, in this article we'll show you how to sample multiple videos through a feature called Videowall.
