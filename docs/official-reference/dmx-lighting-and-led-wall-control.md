# DMX, Lighting, and LED Wall Control

!!! info "Official source"

    Summary based on Assimilate's [02 - Live FX User Guide](https://www.assimilatesupport.com/akb/KnowledgebaseArticle51035.aspx).

The official guide covers DMX controllers, fixture setup, fixture control, color sampling, other DMX controls, and LED wall control.

## DMX and fixtures

Live FX can control lighting through DMX-style protocols, including Art-Net, USB devices, and sACN. Fixture setup includes managing fixtures, configuring their channels, and patching control to the appropriate universe/channel layout.

## Image-based lighting

Fixture color can be driven manually or sampled from video content. This connects lighting to the same media and color-managed workflows used for LED walls and green-screen backgrounds.

## LED wall control

LED wall control is part of the broader live environment: media output, wall mapping, projection, and lighting should be planned together so the stage behaves consistently.
