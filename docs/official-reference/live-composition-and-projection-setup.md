# Live Composition and Projection Setup

!!! info "Official source"

    Summary based on Assimilate's [02 - Live FX User Guide](https://www.assimilatesupport.com/akb/KnowledgebaseArticle51035.aspx).

The official guide treats live composition setup, live projection setup, and composition assembly as related but distinct workflows.

## Live composition setup

Live composition combines live capture, media layers, grades, keying, tracking, and metadata. Record options can capture a finished composite or source captures for later offline/online work.

## Live projection setup

Live projection setup includes choosing sources, configuring capture, building projection composition structure, and using Stage Manager/camera data for wall projection and set-extension workflows.

## Composition assembly

The official guide describes a path from live composition to offline composition and then to online composition. Source captures, sidecar metadata, and recorded live link animation data are used to reconstruct or refine the final result after production.
