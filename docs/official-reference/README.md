# Official Reference

!!! info "Official source"

    This section summarizes the official Assimilate Live FX User Guide: [02 - Live FX User Guide](https://www.assimilatesupport.com/akb/KnowledgebaseArticle51035.aspx).

This reference section is the authoritative layer for Live FX concepts and terminology. The rest of this site remains focused on practical workflows, example projects, and studio-oriented notes.

Use these pages when you need to understand how Live FX is organized, what a panel or node is meant to do, or how official behavior maps onto a tutorial.
