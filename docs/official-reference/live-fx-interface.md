# Live FX Interface

!!! info "Official source"

    Summary based on Assimilate's [02 - Live FX User Guide](https://www.assimilatesupport.com/akb/KnowledgebaseArticle51035.aspx).

The official guide frames Live FX as a virtual production toolset for live compositing, previsualization, LED wall projection, metadata capture, recording, and image-based lighting.

## Main areas

The **Construct** tab is where projects, timelines, shots, media, versions, metadata, and shot setup are managed. This is the organizational side of the workflow before entering a live setup.

The **Live FX** tab is the live operation area. The official guide groups key live functions here: Live Links, Stage Manager, calibration, scene/take metadata, DMX, performance monitoring, online composition tools, and quick rendering utilities.

The **Player Controls** include live review and record behavior. The guide calls out auto/record behavior, cueing, record format, alpha-capable recording formats, and the choice between recording a full composite or separate live source captures.
